TO INSTALL THE FILTER

1) Select one of the filter.files:

2) Paste the file into the following folder:

%userprofile%/Documents/My Games/Path of Exile

INGAME: Escape -> Options -> UI -> Scroll down -> Select NeverSink.filter

Done. Enjoy. There are different filterversions you can use:
Full=all features, Personal=my settings, MF=Magic Find (filters rares a lot) and Slick=Less Colors (more GGG-styled)
